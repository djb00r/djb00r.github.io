How to Corrupt and Load a Scratch Project

  1. In Scratch: File → Save to your computer (gets .sb3).
  2. In Explorer/Finder: rename .sb3 to .zip (enable file extensions if needed).
  3. Load the .zip here and click Corrupt Project.

Load the corrupted project in Scratch

  1. Click "Download Corrupted Project".
  2. Rename the downloaded file from .zip to .sb3.
  3. Open the .sb3 in Scratch. Done.
